# 서울 청년 정책 추천 서비스 - 프로젝트 완성 요약

## 📋 프로젝트 개요

서울시 청년(만 19~34세)을 위한 맞춤형 정책 추천 웹 애플리케이션입니다.
간단한 10개 설문에 답하면 주거·소득·교통·자산형성 분야의 청년 지원 정책을 자동 추천받을 수 있습니다.

## ✅ 완성된 기능

### 1. 백엔드 (FastAPI)
- ✅ `/recommend` API 엔드포인트 구현
- ✅ rules.yaml 기반 정책 규칙 엔진
- ✅ 소득 기준 자동 계산 (2026년 중위소득 기준)
- ✅ 보증금 월세 환산액 계산 로직
- ✅ K-Pass vs 기후동행카드 최적 추천 알고리즘
- ✅ 중복 지원 제한 로직
- ✅ CORS 설정 (프론트엔드 연동)

### 2. 프론트엔드 (React + TypeScript + Tailwind)
- ✅ IntroPage: 토스 스타일의 깔끔한 인트로 페이지
- ✅ SurveyPage: 10개 질문 단계별 설문
  - 휠 선택기 (나이, 소득, 보증금, 월세, 교통 이용)
  - 카드 선택 (거주지, 주택 소유, 가구 형태, 취업 상태, 학생 여부)
  - 진행률 바
- ✅ ResultPage: 추천 결과 카드 UI
  - 카테고리별 배지
  - 추천 이유 표시
  - 예상 절감 금액
  - 신청하기 버튼
- ✅ WheelPicker: iOS 스타일 휠 선택 컴포넌트
- ✅ Zustand 상태 관리
- ✅ React Router 페이지 라우팅

## 🚀 실행 방법

### 백엔드 실행
```bash
cd seoul-youth-policies/backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### 프론트엔드 실행
```bash
cd seoul-youth-policies/frontend
npm install
npm start
```

브라우저에서 http://localhost:3000 접속

## 📝 주요 알고리즘

### 1. 보증금 월세 환산액 계산
```
합산액 = 월세 + (보증금 × 0.05 ÷ 12)
```

### 2. 소득 기준 판정
```
2026년 1인 가구 중위소득 150% = 3,846,357원
```

### 3. 교통비 최적화
```
월 이용액 > 77,500원 → 기후동행카드 추천 (55,000원 무제한)
월 이용액 ≤ 77,500원 → K-Pass 추천 (30% 환급)
```

## 🎯 테스트 시나리오

### 시나리오 1: 월세 청년
- 25세, 서울 거주, 독립 1인 가구
- 정규직, 월 250만원
- 보증금 3,000만원, 월세 50만원
- 대중교통 월 44회

**예상 결과**: 월세 지원, K-Pass 등 3~4개 정책 추천

### 시나리오 2: 미취업 청년
- 27세, 서울 거주, 독립 거주
- 미취업, 월 소득 50만원 이하
- 대중교통 월 10회

**예상 결과**: 청년수당, 국민취업지원제도 추천
